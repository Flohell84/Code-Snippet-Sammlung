# Code-Snippet-Sammlung

Eine moderne Web-App zur Verwaltung von Code-Snippets mit Suche und Kategorien.

## Features
- Snippets speichern mit Titel, Beschreibung, Code, Sprache und Kategorie
- Suchen und Filtern nach Sprache und Kategorie
- Bearbeiten und Löschen von Snippets

## Tech-Stack
- Frontend: React
- Backend: Node.js/Express
- Datenbank: MongoDB

## Projektstruktur
- `client/` – React-Frontend
- `server/` – Node.js/Express-Backend
